# HM-Sales-Dashboard-PowerBI
Power BI Dashboard analyzing H&amp;M sales sample data using DAX and data modeling
The data is sourced from Roberd Kwiatkowski on Kaggle. Check https://www.kaggle.com/code/datark1/detailed-eda-understanding-h-m-data
The transaction data was filtered to include only July 2019 due to considerable size of the dataset. However, the dashboard is scalable to include the entire transaction dataset available on Kaggle.
